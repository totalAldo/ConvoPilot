Getting Started

How to use the app

  1. Open ConvoPilot and complete the first-run legal setup.
  2. Choose your legal region manually, or use location to suggest a region if you want help identifying it.
  3. Keep transcription on device unless you intentionally opt into cloud processing in Settings.
  4. Start a session before the conversation begins.
  5. Watch the live transcript, mood timeline, highlights, and analysis cues during the conversation.
  6. Stop the session when finished. Summaries are only stored if you enabled saving in Settings.
Permissions

What you may need to allow

  • Microphone: required for live transcription.
  • Speech recognition: required for Apple speech transcription.
  • Notifications: optional and only used for local analysis prompts when enabled.
  • Location: optional and only used to suggest a legal region if you request it.
Troubleshooting

Common issues

I started a session but do not see live transcription.

Check microphone and speech permissions first. Then confirm your current locale is supported for on-device transcription on the device you are using.

The transcript does not seem to include the other person on a call.

That can happen on iPhone. ConvoPilot version 1 is microphone-first and does not guarantee two-sided capture from Phone, FaceTime, Zoom, Bluetooth audio, or the earpiece path.

I see an on-device or cloud badge. What does it mean?

The badge shows where transcription for the current session is happening. On-device is the default. Cloud processing should only appear if you explicitly allowed it and a cloud mode is actively in use.

Can ConvoPilot give legal advice about recording a call?

No. The app provides general guidance only. You are responsible for complying with the laws that apply where you and the other participants are located.

Are summaries or raw audio saved automatically?

Raw audio is not saved. Session summaries are only stored on device if you turned summary saving on in Settings.
